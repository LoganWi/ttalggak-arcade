# /add-game: 신규 게임 추가 워크플로우

새로운 게임을 추가할 때는 다음 양식을 사용하여 Antigravity(AI)에게 요청합니다.

### 📋 신규 게임 등록 신청 양식
1. **게임 제목**: (예: 부장님의 습격)
2. **한 줄 설명**: (예: 부장님의 잔소리를 클릭으로 피하세요!)
3. **태그**: (예: #순발력, #스트레스해소)
4. **테마 컬러**: (예: #FF5733 / 카드 하단 PLAY 글자 색상)
5. **소스 코드**: (HTML/JS/CSS 코드를 파일별로 또는 전체 복합)
6. **썸네일**: (이미지 업로드 또는 'AI 생성 요청')

### ⚙️ Antigravity의 작업 자동화 단계
1. **파일 배치**: `public/games/[game-id]/` 폴더 생성 및 소스 코드 저장.
2. **코드 최적화**: 게임 코드 내 API 경로(`/api-playcount.js` 등)를 절대 경로로 자동 수정.
3. **메인 로비 업데이트**: `src/app/page.js`의 `gamesData` 배열에 신규 게임 정보 추가 및 3D 카드 UI 적용.
4. **썸네일 처리**: 이미지가 없는 경우 게임 내용에 맞춰 AI로 고퀄리티 이미지 생성 및 `public/assets/`에 저장.
5. **연동 확인**: GA 추적 및 플레이 카운트 기능이 로컬에서 정상 작동하는지 확인.
6. **배포**: 사용자의 최종 승인 후 `/deploy` 워크플로우를 실행하여 실서버에 반영.
